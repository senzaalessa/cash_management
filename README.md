# cash_management
Il progetto consiste nella realizzazione di un web service per la gestione delle spese che ha l’obiettivo di permettere agli utenti di tracciare, organizzare e analizzare le proprie uscite in modo semplice e veloce. Il web service permetterà di registrare nuove spese, consultare lo storico e ottenere riepiloghi desiderati dall'utente.
